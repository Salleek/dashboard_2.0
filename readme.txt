Dashboard 2.0 is our intuitive way of getting people all over more connected to their cars. To run our program however, you will need some specific hardware. See the list below.

Hardware Needed:
1. Raspberry Pi 3 B+ preferred or any Raspberry Pi with a built in bluetooth adapter. Heat sinks are highly recommended.
2. Bluetooth ELM327 OBDII Adapter
3. 1024*600 resolution screen

Once you have acquired all these components and have your Raspberry Pi running on Raspbian, please do the following steps.

1. Make sure to connect the Raspberry Pi to the OBD Dongle via Bluetooth.
2. Run the following command: pip3 install time
3. Run the following command: pip3 install pygame
4. Run the following command: pip3 install obd

Once all this is done, open the directory with all the Dashboard 2.0 files run the python file dashboard.py and Dashboard 2.0 will launch.
